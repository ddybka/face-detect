Необходимо реализовать распознавание стульев: https://google.github.io/mediapipe/images/objectron_data_annotation.gif 
в идеале получить координаты (x, y, z) одной из вершин описывающего стул куба в пространстве по
документации https://google.github.io/mediapipe/solutions/objectron.html под виндой, координаты 
вершин описывающих стулья кубов выводим в консоль.

Можно распознавать туфли, стулья, чашки и фотоаппараты. 
Для этого надо в коде параметр model_name установить в 'Shoe', 'Chair', 'Cup', или 'Camera' 
соответственно. Несколько из них распознавать в принципе можно, но для этого нужно будет отдельно 
запускать соответствующую модель для каждого кадра.
