1) Заходим в пуск, в поиске пишем cmd, нажимаем правой кнопкой мыши на появившуюся иконку и выбираем "Запуск от имени администратора"
2) Вбиваем в открывшееся окно pip install opencv-python ждем пока завершится загрузка (лучше сразу ставить pip install opencv-contrib-python)
3) Аналогично pip install mediapipe
Если с 2) и 3) возникли проблемы то скорее всего у вас отсутствует pip или путь к нему не указан в PATH. Проще всего это пофиксить так:
Удаляем Python(панель управления->установка и удаление программ), идем на сайт www.python.org/downloads скачиваем последнюю версию ОБЯЗАТЕЛЬНО СТАВИМ ГАЛОЧКУ "Add Python 3.9 to PATH"
Теперь всё должно работать.
