Эта задача заключается в том, чтобы написать приложение на py под винду для повторения
этой задачи: https://www.pyimagesearch.com/2015/09/14/ball-tracking-with-opencv/
видео: https://youtu.be/dKpRsdYSCLQ
гит: https://github.com/hemkum/ball-tracking-with-opencv

Ну в этой задаче нужно только скопировать код с гитхаба и установить библиотеки numpy, matplotlib, pandas (т.е. просто в командной строке написать pip install *название_библиотеки* 3 раза). 
Код работает только для мячей одного цвета, в скрипте - для зеленого, чтобы изменить цвет нужно настроить параметры greenLower и greenUpper.
